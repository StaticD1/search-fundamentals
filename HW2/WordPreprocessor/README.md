## HTML Text Processor & Lemmatizer

Приложение на базе Spring Boot для предварительной обработки текста в задачах информационного поиска. Программа
выполняет очистку HTML-контента, токенизацию, фильтрацию и лемматизацию русских слов с агрегацией результатов по всей
коллекции документов.

### Особенности

* Агрегация данных: Результаты обработки всех страниц собираются в два общих файла.
* Очистка HTML: Использование библиотеки Jsoup для корректного удаления тегов, скриптов и стилей.
* Токенизация: Разбиение текста на отдельные слова с удалением дубликатов.
* Фильтрация:
    * Удаление стоп-слов (союзы, предлоги) на базе стандартного списка Apache Lucene.
    * Удаление "мусора" (слова с цифрами, обрывки разметки) через регулярные выражения.
* Лемматизация: Приведение слов к основе с использованием алгоритма Стеммера Портера (Snowball) для русского языка.

## Установка и запуск

### Предварительные требования

* JDK 17 или выше
* Apache Maven

### Шаг 1: Сборка

Склонируйте проект и выполните команду:

```bash
mvn clean install
```

### Шаг 2: Подготовка данных

Создайте папку data/input в корне проекта.
Поместите туда файлы .txt, содержащие HTML-разметку.

### Шаг 3: Запуск

```bash
mvn spring-boot:run
```

### Результаты работы

После завершения работы программы в папке data/output создаются два файла:

#### 1. tokens.txt

Список всех уникальных токенов, встретившихся во всех документах.

Формат:

```text
<токен>
<токен>
...
```

Пример:

```text
алгоритм
веб
данные
поиск
система
```

#### 2. lemmas.txt

Группировка токенов по леммам (основам слов) по всей коллекции документов.

Формат:

```text
<лемма> <токен1> <токен2> ...
```

Пример:

```text
алгоритм алгоритм алгоритме алгоритма
документ документ документа документах
поиск поиск поиска
```

### Алгоритм обработки

1. Извлечение текста: Библиотека Jsoup удаляет HTML-теги, оставляя чистый текст.
2. Токенизация: Текст разбивается по не-буквенным символам.
3. Очистка:
    * Приведение к нижнему регистру.
    * Удаление слов с цифрами.
    * Удаление стоп-слов.
4. Стемминг: Вычисление основы слова (леммы).
5. Агрегация: Сбор данных в общие коллекции.
6. Сохранение: Запись итоговых файлов.
