<p><b>Стрельба из лука</b>&#160;— искусство или практическое умение по стрельбе в <a href="/wiki/%D0%A6%D0%B5%D0%BB%D1%8C" title="Цель">цель</a> (либо на дальность) с использованием <a href="/wiki/%D0%9B%D1%83%D0%BA_(%D0%BE%D1%80%D1%83%D0%B6%D0%B8%D0%B5)" title="Лук (оружие)">лука</a> и стрел.
</p><p>Искусство стрельбы из лука возникло в эпоху позднего <a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%B5%D0%BE%D0%BB%D0%B8%D1%82" title="Палеолит">палеолита</a> либо раннего <a href="/wiki/%D0%9C%D0%B5%D0%B7%D0%BE%D0%BB%D0%B8%D1%82" title="Мезолит">мезолита</a>. Старейшие находки, связанные со стрельбой из лука, датируются <a href="/wiki/IX_%D0%B2%D0%B5%D0%BA_%D0%B4%D0%BE_%D0%BD._%D1%8D." title="IX век до н. э.">IX</a>—<a href="/wiki/VIII_%D0%B2%D0%B5%D0%BA_%D0%B4%D0%BE_%D0%BD._%D1%8D." title="VIII век до н. э.">VIII&#160;веками до&#160;н.&#160;э.</a> Исторически стрельба из лука использовалась для охоты и боевых действий. Лук использовался с такой целью практически повсеместно. Единственным <a href="/wiki/%D0%A0%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD" title="Регион">регионом</a>, где искусство стрельбы из лука не было известно вплоть до <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX&#160;века</a>, является <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F_%D0%B8_%D0%9E%D0%BA%D0%B5%D0%B0%D0%BD%D0%B8%D1%8F" title="Австралия и Океания">Австралия и Океания</a>. В крупных боевых действиях лук последний раз использовался в начале XIX&#160;века.
</p><p>В настоящее время является <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D1%8C%D0%B1%D0%B0_%D0%B8%D0%B7_%D0%BB%D1%83%D0%BA%D0%B0_(%D0%B2%D0%B8%D0%B4_%D1%81%D0%BF%D0%BE%D1%80%D1%82%D0%B0)" title="Стрельба из лука (вид спорта)">одним из видов спорта</a>.
</p><p>В некоторых странах стрельба из лука является достаточно популярным видом отдыха и развлечения, несмотря на потенциальную опасность. На специально оборудованных стрельбищах устанавливают мишени, чаще всего из спрессованной соломы. (См., например, <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D1%8C%D0%B1%D0%B0_%D0%B8%D0%B7_%D0%BB%D1%83%D0%BA%D0%B0_%D0%B2_%D0%91%D1%83%D1%82%D0%B0%D0%BD%D0%B5" title="Стрельба из лука в Бутане">Стрельба из лука в Бутане</a>.)
</p><p>Дистанция для стрельбы не регламентируется, но можно использовать расстояние, принятое у профессиональных охотников из лука&#160;— 20 <a href="/wiki/%D0%AF%D1%80%D0%B4" title="Ярд">ярдов</a>.
</p>
